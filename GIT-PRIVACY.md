# Правила обмена с Git

> Зафиксировано: 10.07.2026 · обновлено: 10.07.2026

## Принцип

**В Git — код и санитизированный снимок.**  
**Полные данные по клиентам — локально в `CURSOR/` и в чате по запросу.**

---

## Запрещено выгружать в Git / GitHub

| Категория | Примеры |
| --- | --- |
| Телефоны | `+7 925 …`, любые номера |
| Почта и мессенджеры | email, Telegram, @username |
| Клиенты | Названия компаний, бренды, партнёры |
| Суммы | ₽, $, «15K/мес», цены, подписки |
| Пути и файлы | `cursor_root`, пути к `BACKLOG.md` |
| Переговоры | Журналы встреч, комментарии с деталями сделок |

---

## Разрешено в Git (исключения)

| Категория | Примеры |
| --- | --- |
| **ФИО** | Ответственные: Рафаэль, Андрей; имена в поле `assignee` |
| Задачи | ID, приоритет, статус, текст (без телефонов и сумм) |
| Агрегаты | Счётчики, коды проектов `P-01`, домены `3D` |
| События | Даты, типы, статусы (без названий клиентов) |

---

## Локальные файлы (не в Git)

| Файл | Содержимое |
| --- | --- |
| `data/planner.local.json` | Полный снимок с клиентами |
| `web/data/planner.local.json` | Копия для локального UI |
| `CURSOR/**/BACKLOG.md` | Источник правды |
| `CURSOR/**/КАЛЕНДАРЬ.md` | Календарь |

---

## Пайплайн

```
BACKLOG.md / КАЛЕНДАРЬ.md (локально)
        ↓ sync_data.ps1
planner.local.json     ← полные данные (НЕ в Git)
planner.json + docs/   ← санитизированный снимок (ФИО ✓)
        ↓ auto_sync.ps1
git push → GitHub Pages
```

---

## Для агентов

- ФИО в Git — **разрешено** (поле assignee, имена людей)
- Телефоны, почты, клиенты, суммы — **запрещено**
- Не коммитить `planner.local.json`
- Полные данные по клиентам — в чате по запросу
