# Правила обмена с Git

> Зафиксировано: 10.07.2026 · обновлено: 10.07.2026

## Принцип

**В Git — код и рабочий снимок без контактных реквизитов.**  
**Телефоны и суммы — только локально в полном `planner.local.json`.**

---

## Разрешено в Git / GitHub

| Категория | Примеры |
| --- | --- |
| **ФИО** | Контакты клиентов, ответственные, сотрудники |
| **Компании** | Названия клиентов, бренды, организации |
| **Задачи** | Тексты, приоритеты, статусы, assignee |
| **События** | Даты, клиент, тема, тип |
| **Агрегаты** | Счётчики, домены `3D`, `PROVANCE` |

---

## Запрещено в Git

| Категория | Примеры |
| --- | --- |
| Телефоны | `+7 …`, любые номера |
| Почта | email, @username как контакт |
| Суммы | ₽, $, подписки, цены, «15K/мес» |
| Локальные пути | `cursor_root`, абсолютные пути ПК |

---

## Локально (не в Git)

| Файл | Содержимое |
| --- | --- |
| `data/planner.local.json` | Полный снимок с телефонами |
| `CURSOR/**/BACKLOG.md` | Источник правды |

---

## Пайплайн

```
BACKLOG.md / КАЛЕНДАРЬ.md
        ↓ sync_data.ps1
planner.local.json  ← полные данные (НЕ в Git)
planner.json        ← имена и компании ✓, телефоны и суммы ✗
        ↓ auto_sync.ps1 → GitHub Pages
```
