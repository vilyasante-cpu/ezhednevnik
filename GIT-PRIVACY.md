# Правила обмена с Git

> Зафиксировано: 10.07.2026

## Принцип

**В Git — только код и обезличенная статистика.**  
**Вся рабочая информация — локально в `CURSOR/` и показывается вам в чате по запросу.**

---

## Запрещено выгружать в Git / GitHub

| Категория | Примеры |
| --- | --- |
| Контакты | ФИО, должности, «ЛПР», участники переговоров |
| Телефоны | `+7 925 …`, любые номера |
| Почта и мессенджеры | email, Telegram, @username |
| Клиенты | Названия компаний, бренды, имена партнёров |
| Суммы | ₽, $, «15K/мес», цены, подписки, скидки |
| Пути и файлы | `cursor_root`, пути к `BACKLOG.md` |
| Содержание | Тексты задач, итоги встреч, комментарии с именами |

---

## Разрешено в Git

- Исходный код UI (`web/`)
- Скрипты (`scripts/`)
- Документация проекта (без клиентских данных)
- **Обезличенный** `planner.json`:
  - количество клиентов, задач, событий
  - домены проектов (`3D`, `PROVANCE`)
  - коды вида `P-01`, `P-02` вместо имён
  - даты и типы событий без привязки к клиенту

---

## Локальные файлы (не в Git)

| Файл | Содержимое |
| --- | --- |
| `data/planner.local.json` | Полный снимок |
| `web/data/planner.local.json` | Копия для локального UI |
| `CURSOR/**/BACKLOG.md` | Источник правды |
| `CURSOR/**/КАЛЕНДАРЬ.md` | Календарь |

Файлы `*.local.json` перечислены в `.gitignore`.

---

## Пайплайн

```
BACKLOG.md / КАЛЕНДАРЬ.md (локально)
        ↓ sync_data.ps1
planner.local.json          ← полные данные (НЕ в Git)
planner.json + docs/        ← только санитизированный снимок
        ↓ auto_sync.ps1
git push → GitHub Pages (без персональных данных)
```

---

## Для агентов

- Показывать полные данные пользователю **только в чате** и **только по запросу**
- Не коммитить `planner.local.json`
- Не отключать санитизацию без явного указания пользователя
- При обновлении скриптов sync — сохранять правило приватности
